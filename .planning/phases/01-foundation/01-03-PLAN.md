Use div-based D3 rendering (NOT SVG) per Research Open Question 3 recommendation. This makes text wrapping and CSS styling simpler.

**StackPanel** (`src/viz/StackPanel.ts`):
- Constructor takes `container: HTMLElement`
- `render(frames: StackFrame[]): void`
- On render: `d3.select(container).selectAll('*').remove()` (Pitfall 3 prevention)
- If frames empty: show placeholder div with muted text "Write code and click Run"
- Render frames in reverse order (most recent at top = top of stack)
- Each frame: div with class `stack-frame`, border-left color `var(--color-stack-border)`
- Frame header: `frame.name()` bold
- Each local: row showing `type name = value` and hex address `0x${address.toString(16)}`
- Use CSS classes, colors via `var(--color-*)` custom properties

**HeapPanel** (`src/viz/HeapPanel.ts`):
- Constructor takes `container: HTMLElement`
- `render(blocks: HeapBlock[]): void`
- Clear container before render
- If blocks empty: show placeholder
- Each block: div with class `heap-block`
- Show: hex address `0x${address.toString(16)}`, size in bytes, status badge
- Status colors: allocated = green-ish, freed = red-ish/dimmed, leaked = warning
- If label exists, show it next to address

Write tests using jsdom — create a container div, instantiate panel, call render with fixture data, assert DOM contains expected elements via `container.querySelectorAll()`.
