**Update the `render` method signature** to accept an optional second parameter for diff:

```typescript
render(blocks: HeapBlock[], prevBlocks: HeapBlock[] = []): void
```

**Inside render(), build a changed-address set for diff highlighting:**

```typescript
const changedAddrs = new Set<number>();
for (const block of blocks) {
  const prev = prevBlocks.find(p => p.address === block.address);
  if (prev && prev.status !== block.status) changedAddrs.add(block.address);
}
```

**Add `data-address` to each block container** (on the `.heap-block` div):

```typescript
.attr('data-address', String(block.address))
```

**Add leaked block background tint:** When `block.status === 'leaked'`, set background to `rgba(212, 172, 13, 0.15)` instead of the default `var(--color-bg-panel)`.

**Add LEAK badge** in the header row, after the address (and label if present), when `block.status === 'leaked'`:

```typescript
if (block.status === 'leaked') {
  headerRow
    .append('span')
    .attr('class', 'heap-leaked-badge')
    .style('background', '#d4ac0d')
    .style('color', '#fff')
    .style('font-size', '11px')
    .style('font-weight', '700')
    .style('padding', '1px 6px')
    .style('border-radius', '10px')
    .style('text-transform', 'uppercase')
    .text('LEAK');
}
```

**Add diff highlighting:** If `changedAddrs.has(block.address)`, add `.attr('data-changed', 'true')` to the block div.
