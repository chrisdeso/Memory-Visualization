**Update the `render` method signature** to accept an optional second parameter:

```typescript
render(frames: StackFrame[], prevFrames: StackFrame[] = []): void
```

**Inside render(), before iterating frames, build a changed-address set:**

```typescript
const changedAddrs = new Set<number>();
const prevLocals = prevFrames.flatMap(f => f.locals);
for (const frame of frames) {
  for (const v of frame.locals) {
    const prev = prevLocals.find(p => p.address === v.address);
    if (prev && prev.value !== v.value) changedAddrs.add(v.address);
  }
}
```

**Refine each frame rendering** (inside the `reversed.forEach` loop):

1. Add top border to `.stack-frame` for frame boundary:
   - `.style('border-top', '1px solid #d0d0d0')`

2. Replace the existing frame header with a styled label per UI-SPEC:
   - Class: `stack-frame-name`
   - Font size: 11px, weight 700, color `var(--color-stack-border)`, text-transform uppercase, letter-spacing 0.06em
   - Text: `frame.name + '()'`

3. Add return address row after frame name:
   - Class: `stack-return-addr`
   - Style: flex row, font-size 11px
   - Label span: "return addr:" in color `var(--color-text-muted)`, weight 400
   - Value span: hex address in color `var(--color-address)`, font-family monospace
   - When `frame.returnAddr === 0`, display value as `"\u2014"` (em dash) instead of hex
   - Otherwise display `"0x" + frame.returnAddr.toString(16)`

4. On each local variable row (`.stack-local-row`):
   - Add `.attr('data-address', String(v.address))`
   - If `changedAddrs.has(v.address)`, add `.attr('data-changed', 'true')` and `.style('background', 'var(--color-diff-highlight)')`
